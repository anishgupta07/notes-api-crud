# 📝 Notes API (Spring Boot)

## 🚀 Features
- Create Note
- Get All Notes
- Update Note
- Delete Note
- Auto timestamp (createdAt, updatedAt)

## 🛠 Tech Stack
- Java
- Spring Boot
- REST API

## 📌 Endpoints

### POST /notes
Create a note

### GET /notes
Get all notes

### PUT /notes/{id}
Update note

### DELETE /notes/{id}
Delete note

## 📷 Sample JSON

{
  "title": "Test",
  "description": "My first note"
}
